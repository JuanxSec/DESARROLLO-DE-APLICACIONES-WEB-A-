Inicio

JuansecCTI es un proyecto personal enfocado en compartir boletines, noticias y alertas de ciberinteligencia. Su objetivo es presentar información útil sobre ciberseguridad de una forma sencilla, clara y fácil de entender para empresas, entidades y personas interesadas en proteger sus activos digitales.

Quiénes Somos

JuansecCTI es un proyecto creado con el objetivo de informar sobre temas importantes de ciberinteligencia y ciberseguridad. El sitio busca recopilar noticias, boletines y alertas que puedan aportar valor a empresas y entidades que necesitan conocer los riesgos digitales actuales. La idea principal es explicar amenazas, vulnerabilidades, filtraciones de información y recomendaciones de seguridad de una manera sencilla, sin usar un lenguaje demasiado complicado. Este proyecto también permite aplicar los conocimientos aprendidos en la asignatura Desarrollo de Aplicaciones Web, utilizando una estructura correcta de HTML5, etiquetas semánticas y contenido organizado. JuansecCTI representa un espacio informativo básico, pero útil, para aprender y compartir información relacionada con la seguridad digital.
